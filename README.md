# Home Page - Django

Este projeto foi desenvolvido para criação de um portfólio pessoal.

## Framework e bibliotecas utilizadas
* Linguagem Python
* Framework Django 3.2.9
* Biblioteca python-decouple 3.5
* Biblioteca psycopg2-binary 2.9.2
